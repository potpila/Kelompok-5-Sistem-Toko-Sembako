# Aplikasi-Manajemen-Toko-Sembako
Aplikasi kasir dan manajemen stok berbasis Java Swing dan MySQL untuk memenuhi tugas proyek kelompok Pemrograman 1.

Langkah Penggunaan:
1. Unduh Proyek: Klik tombol Code berwarna hijau di kanan atas repositori ini, lalu pilih Download ZIP. Ekstrak file ZIP tersebut di komputer Anda.
2. Aktifkan XAMPP: Buka XAMPP Control Panel, kemudian klik Start pada bagian Apache dan MySQL.

3. Siapkan Database:
   - Buka browser dan masuk ke (http://localhost/phpmyadmin/).
   - Buat database baru (pastikan namanya "db_sistem_sembako").
   - Pilih database tersebut, klik menu Import, lalu pilih file "db_sistem_sembako.sql" yang berada di dalam folder database proyek ini.

4. Jalankan Aplikasi:
   Buka folder "Aplikasi_Siap_Pakai_JAR", lalu klik dua kali pada file "Proyek_Kelompok.jar" untuk menjalankan aplikasi tanpa perlu membuka NetBeans.
